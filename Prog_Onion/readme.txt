En el presente repositorio encontraras la documentación necesaria para montar un sistema de IoT en encendido y apagado de un led utilizando la placa Onion Omega 2. 

El proyecto se encuentra dividido en 3 partes:
1. Configuración y programación del Onion omega2
2. Desarrollo de App nativa Android.
3. Desarrolla de App web.
